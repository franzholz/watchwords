============
Installation
============

Use the TYPO3 Extension Manager or use composer to install this extension.

See :ref:`t3install:start`.

Then insert a plugin Watchword (watchwords_watch) where you want to show it.
Alternatively you can use a setup instead of a plugin.
Insert the basic setup provided by this extension by adding **"Display daily Christian Watchwords (watchwords)"** in the Template module setup under "Include static (from extensions)".
